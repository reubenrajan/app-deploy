JMeter Aggregate Report columns (File → Save Table Data)
service — deployment name matching a selected microservice (add this column to your JMeter CSV in CI)
Label — transaction / sampler label (e.g. POST /payments)
# Samples — total number of requests in the interval
Average — mean response time (ms)
Median — 50th percentile response time (ms)
90% Line — 90th percentile response time (ms)
95% Line — 95th percentile response time (ms)
99% Line — 99th percentile response time (ms)
Error % — HTTP error percentage (0.00%–100.00%)
Throughput — requests/sec
Min — minimum response time (ms)
Max — maximum response time (ms)
Received KB/sec — received bandwidth
Sent KB/sec — sent bandwidth
phase — pre / chaos / post (add this column to tag test phases)
